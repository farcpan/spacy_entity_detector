# Spacy Sample on WSL/Ubuntu

## venv

```bash
$ python -m venv .venv
$ source .venv/bin/activate
```

---

## Installation

```bash
(venv)$ pip install spacy ja-ginza
```

### Other Models

```bash
(venv)$ python -m spacy download ja_core_news_md
```
